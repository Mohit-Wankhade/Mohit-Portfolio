# Mohit Wankhade Portfolio

Static portfolio website for GenAI Engineer / AI-ML Engineer / Software Engineer roles.

## What's inside

- `index.html` — main portfolio page
- `styles.css` — responsive dark AI-themed styling
- `script.js` — menu + scroll animations
- `assets/` — profile image, logo, banner, resume PDF
- `vercel.json` — Vercel-ready config

## Deploy on Vercel

### Option 1: Drag and drop

1. Go to Vercel Dashboard.
2. Use Vercel Drop / Upload project.
3. Upload this folder.

### Option 2: GitHub import

1. Create a new GitHub repository named `mohit-portfolio`.
2. Upload these files to the repository root.
3. Go to Vercel → New Project → Import Git Repository.
4. Select the repository and deploy.

For a static site, no build command is required.
